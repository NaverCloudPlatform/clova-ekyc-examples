data class ClovaIdCardDetectorOption @JvmOverloads constructor(val cardRatio: Float = 1.5858f, val targetAngle: Int = 90, val angleOffset: Int = 5, val minimumSize: Float = 0.65f, val checkSideDetected: Boolean = false, val checkCameraShaky: Boolean = false, val checkImageBlurry: Boolean = false, val checkImageDark: Boolean = false, val checkGlareDetected: Boolean = false, val blurThreshold: Float = 0.13f, val darkThreshold: Int = 7, val glarePercentage: Float = 0.05f, val faceLocation: ClovaIdCardDetectorOption.IdFaceLocation = IdFaceLocation.ANY_WHERE, val accumulatedFrameCount: Int = 10)

The container for options of the ClovaIdCardDetector.

Parameters

cardRatio

Target width height ratio of card to detect, default is 1.5858 International Card Ratio (ID-1) (total card area) ROUND (85.60/53.98)

targetAngle

Target angle of card to detect, default is 90.

angleOffset

Offset of angle to detect, default is 5. targetAngle - angleOffset <= detect range <= targetAngle + angleOffset The possible angle range is 70-110 degrees.

minimumSize

The minimum size of a bounding rectangle to detect, as a proportion of the smallest dimension.

checkSideDetected

Whether to detect each side line of the card.

checkCameraShaky

Whether to detect camera shaking.

checkImageBlurry

Whether to detect blurry image.

checkImageDark

Whether to detect dark image.

checkGlareDetected

Whether to detect glare image.

blurThreshold

Blur judgment threshold.

darkThreshold

Dark judgment threshold.

glarePercentage

Indicates what percentage of the card area should be occupied by light reflection to determine as an error.

faceLocation

The location of the face in id card.

accumulatedFrameCount

Total number of frames to be accumulated to determine the best id card shot.
