Parameters

cardRatio

Target width height ratio of card to detect, default is 1.5858 International Card Ratio (ID-1) (total card area) ROUND (85.60/53.98)

targetAngle

Target angle of card to detect, default is 90.

angleOffset

Offset of angle to detect, default is 5. targetAngle - angleOffset <= detect range <= targetAngle + angleOffset The possible angle range is 70-110 degrees.

minimumSize

The minimum size of a bounding rectangle to detect, as a proportion of the smallest dimension.

checkSideDetected

Whether to detect each side line of the card.

checkCameraShaky

Whether to detect camera shaking.

accumulatedFrameCount

Total number of frames to be accumulated to determine the best card shot.
